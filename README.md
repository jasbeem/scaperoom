# ScapeRoom

Aplicación de escape room multijugador para aula. Los jugadores eligen avatar, escriben su nombre y entran con un código de misión.

## Requisitos
- Node.js (recomendado LTS)

## Instalación
```bash
npm install
```

## Desarrollo
```bash
npm run dev
```

## Build
```bash
npm run build
```
El resultado queda en la carpeta `dist/`.

## Preview
```bash
npm run preview
```

## Notas
- Los audios se sirven desde `public/audio/`.
- Si necesitas una variable de entorno, crea `.env.local` en la raíz.
